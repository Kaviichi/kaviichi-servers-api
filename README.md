#

## Development

- Uses prettier for code formatting, it is recommended to install the VS Code Prettier addon with development.

### Getting Started

- Install packages
    ```
    npm install
    ```
- Start the server with nodemon
    ```
    nodemon src/server.ts
    ```
- Run tests
    ```
    npm run test
    ```
- Build the js files
    ```
    npm run build
    ```
